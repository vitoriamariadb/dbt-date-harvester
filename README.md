# dbt-date-harvester

Ferramenta de parsing e analise de projetos dbt.

## Sobre

Parser e analisador para projetos dbt (data build tool). Extrai metadados de
arquivos YAML e SQL, constroi grafos de dependencia e fornece ferramentas de
analise para projetos dbt.

## Instalacao

```bash
pip install -e .
```

## Uso Basico

```bash
dbt-parser --project-dir /caminho/projeto/dbt
```
